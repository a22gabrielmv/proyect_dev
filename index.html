HTML introduction

25/9/2022

HTML provides structure to the content on the website, such as images, text, or videos. HTML stands for HyperText Markup Language

  • A markup language is a caomputer language that defines the structure and presentation of raw text.
  • In HTML, the computer can interpret raw text that is wrapped in HTML elemnets.
  • HyperText is text displayed on a computer or device that provides access to other text through links, also known as hyperlinks.
